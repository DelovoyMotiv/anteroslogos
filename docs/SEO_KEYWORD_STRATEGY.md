## Technical SEO Requirements

### Schema Markup Additions

**For All New Pages:**
1. Breadcrumb schema
2. Article schema (blog/guides)
3. FAQPage schema where applicable
4. HowTo schema for guides
5. Service schema for service pages

**Example for AI Search Guide:**
```json
{
  "@context": "https://schema.org",
  "@type": "HowTo",
  "name": "How to Optimize for AI Search",
  "description": "Complete guide to AI search optimization",
  "step": [...]
}
```

### URL Structure
- Keep URLs short and keyword-rich
- Use hyphens, not underscores
- Avoid dates in URLs for evergreen content
- Maintain logical hierarchy

- `/geo-vs-seo` ‚úÖ
